# ThreadLocalдҪҝз”ЁеңәжҷҜеҺҹзҗҶ

еңЁйЎ№зӣ®дёӯдё»иҰҒдҪҝз”ЁThreadLocalдҝқеӯҳзәҝзЁӢзҡ„жң¬ең°еҸҳйҮҸпјҢжҜ”еҰӮеҸҜд»ҘеңЁжӢҰжҲӘеҷЁдёӯж”ҫе…Ҙз”ЁжҲ·idпјҢд№ӢеҗҺйғҪеҸҜд»ҘеҫҲж–№дҫҝзҡ„еҸ–еҮәidпјҢдҪҶжҳҜиҰҒжіЁж„ҸеңЁжӢҰжҲӘеҷЁзҡ„еҗҺзҪ®еӨ„зҗҶдёӯдёҖе®ҡиҰҒе°Ҷе…¶removeпјҢдёҚз„¶дјҡеҜјиҮҙеҶ…еӯҳжі„жјҸгҖӮ

иҷҪз„¶еүҚйқўиҜҙзҡ„ж—¶дҪҝз”ЁThreadLocalдҝқеӯҳзәҝзЁӢжң¬ең°еҸҳйҮҸпјҢдҪҶжҳҜжҲ‘жӣҙеҖҫеҗ‘дәҺжҠҠе®ғз§°дҪңдёҖдёӘй’©еӯҗпјҢеҲ©з”Ёе®ғеҸҜд»Ҙе°ҶThreadLocalMapдёӯй’©еҮәжғіиҰҒзҡ„ж•°жҚ®пјҢThreadLocalMapжҳҜThreadзҡ„дёҖдёӘеұһжҖ§пјҢе®ғжҳҜжҮ’еҠ иҪҪзҡ„пјҢеҸӘжңүеҪ“жҲ‘д»¬з¬¬дёҖж¬Ўи®ҝй—®ж—¶жүҚдјҡеҲӣе»әгҖӮ

ThreadLocalMapдёӯзҡ„EntryжҳҜд»ҘеҪ“еүҚзҡ„ThreadLocalдёәkeyпјҢи®ҫзҪ®зҡ„еҖјдёәvalueеӯҳж”ҫзҡ„пјҢеҜ№дәҺдёҚеҗҢзҡ„зәҝзЁӢпјҢдҪҝз”ЁзӣёеҗҢзҡ„ThreadLocalеҺ»иҺ·еҸ–ж•°жҚ®пјҢиҺ·еҸ–еҲ°зҡ„жҳҜThreadиҮӘе·ұзҡ„ThreadLocalMapеҶ…йғЁзҡ„EntryпјҢд№ҹе°ұе®һзҺ°дәҶзәҝзЁӢжң¬ең°еҸҳйҮҸгҖӮеҗ‘еӨ–дҝқжҢҒдәҶеҫҲеҘҪзҡ„е°ҒиЈ…жҖ§гҖӮ
